# はじめに

みなさん初めまして！うっちーです🐱
この度、ポートフォリオサイトを開設しました。
この投稿では、サイトの紹介と今後の展望についてお話しします。

## サイトの特徴

- **技術スタック**: Astro、React、TypeScript、Tailwind CSSを使用しています。
- **デザイン**: シンプルでモダンなデザインかつ、レトロな雰囲気を目指しました。

## サイトの機能

現状、ブログ機能のみ実装しています。
ブログにはMDXを使用しており、JSXコンポーネントを埋め込むことができます。

以下はサンプルのボタンコンポーネントです：

<div class="flex flex-wrap gap-4">
  <Button>デフォルト</Button>
  <Button variant="destructive">注意！</Button>
  <Button variant="secondary">セカンダリ</Button>
  <Button variant="ghost">ゴースト</Button>
  <Button variant="link">リンク</Button>
  <Button variant="outline">アウトライン</Button>
</div>
