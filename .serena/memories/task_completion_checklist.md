# タスク完了時のチェックリスト

タスクを完了したら、以下のコマンドを実行してください。

## 1. フォーマットとリント

```bash
pnpm format
```

このコマンドは以下を実行します:
- Prettierによるコードフォーマット
- ESLintによる自動修正

## 2. ビルド確認

```bash
pnpm build
```

- TypeScriptエラーがないか確認
- ビルドが正常に完了するか確認

## 3. 動作確認（必要に応じて）

```bash
pnpm preview
```

- ビルド結果をローカルでプレビュー

## 4. Git操作（コミット前）

1. `git status` で変更ファイルを確認
2. 不要なファイルが含まれていないか確認
3. `.env` などの機密ファイルがコミットされていないか確認

## 注意事項

- 本番ビルド (`pnpm build`) は必ず通すこと
- リントエラーは修正してからコミット
- 未使用のインポートは削除される（ESLint自動修正）
